# Product Listing Website
A modern and responsive **Product Listing Website** built using **HTML, CSS, and JavaScript**.  
This project showcases products in a professional UI with navigation, filtering, search functionality, and smooth animations.

##  Project Overview
This website is designed to display products in an attractive and user-friendly way.  
Users can browse products, move between items, search by name or owner, and filter products by category.
It is a beginner-friendly front-end project that demonstrates how to combine **HTML structure**, **CSS styling**, and **JavaScript interactivity** into one polished web application.

##  Features
- ✅ Modern and responsive UI
- ✅ Product title, owner, and price display
- ✅ Next product and previous product functionality
- ✅ Search products by name or owner
- ✅ Filter products by category
- ✅ Smooth animations and hover effects
- ✅ Product image showcase
- ✅ Rating, stock status, and product ID display
- ✅ Fully responsive design for desktop and mobile

## Technologies Used
- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
